# 🧬 День науки | Landing & Registration Bot

Простой и стильный одностраничный сайт для регистрации на мероприятие **«День науки»** с бэкендом на FastAPI и интеграцией с Telegram-ботом. 

Проект собирает заявки участников, сохраняет их в базу данных, отправляет уведомления организаторам и позволяет выгружать все данные в Excel прямо из Telegram.

## ✨ Возможности

- 🎨 **Адаптивный лендинг** с красивым градиентом и таймером обратного отсчета до события.
- ✅ **Двойная валидация** данных (на фронтенде и бэкенде).
- 💾 **Локальная база данных** (SQLite) для надежного хранения заявок.
- 🤖 **Telegram-уведомления** о каждой новой регистрации.
- 📊 **Экспорт в Excel** — выгрузка всех участников одной командой в боте.

## 🛠 Стек технологий

| Направление | Технологии |
| :--- | :--- |
| **Frontend** | HTML5, CSS3, Vanilla JS |
| **Backend** | Python 3.10+, FastAPI, Uvicorn |
| **Database** | SQLite3 |
| **Интеграции** | Telegram Bot API, `openpyxl`, `requests` |
